# 程式介紹
此為使用C語言的霍夫曼編碼程式，主要是在Console環境下使用I/O Redirection來將測試檔餵入進執行檔內，或是手動執行執行檔輸入10個符號與機率後，
然後產生每個符號的對應編碼值，以及Entropy ( Hr(2) ) 以及Lav (平均編碼長度)。

# 檔案內容
Huffman.c //主要執行檔
Huffman.h //霍夫曼編碼的標頭檔，放置各種編碼用函式
file1~file5 //測試檔1到5
makefile //makefile編譯腳本

# 環境需求
需要有C的編譯環境

# 使用說明
1. 首先要先進入到console的環境裡，輸入
make
2. 在目錄底下會產生 Huffman.exe，接著輸入
./Huffman.exe < file1
2.1 就會跑出file1測試檔1的編碼結果，例如如下
----------------
Symbol  Prob    CodeWord
A       0.500   0
B       0.400   10
C       0.100   11
Lav =   1.500000
Hr(2) = 1.360964

# 自製測試檔
1. 可以自製測試檔，格式為 “符號 機率”最多可設定10個，例如以下範例
A 0.1
B 0.1
C 0.1
D 0.1
E 0.1
F 0.1
G 0.1
H 0.1
I 0.1
J 0.1
